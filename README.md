# Bright Coffee Shop Sales Analysis

**Junior Data Analyst Submission**  
**Due Date:** 31 March 2026

## Project Overview
Analysis of 149,456 daily transactions from Bright Coffee Shop (Jan–Jun 2023) across three locations to identify:
- Top revenue-generating products  
- Peak sales time periods  
- Sales trends by product and time of day  
- Actionable recommendations for the new CEO

## Repository Contents
- **DataInsights.sql** – Full SQL script used in Databricks  
- **Data/processed_coffee_sales.xlsx** – Cleaned dataset with all pivot tables and Dashboard  
- **Presentation,pdf** – Final CEO presentation  
- **Case Study Flowcart.jpg** – Data architecture diagram  

## Key Insights
- Morning (06:00–09:00) is the strongest period every month  
- Top 3 products (Gourmet brewed coffee, Barista Espresso, Hot chocolate) drive the majority of revenue  
- Evening sales are consistently the weakest  

## How to Run the SQL
1. Open `DataInsights.sql` in VS Code with Databricks connection  
2. Run the Task 2 query to generate the cleaned table  
3. Export as CSV and open in Google Sheets for pivots and charts
